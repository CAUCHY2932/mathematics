字典代替swtich¶

官方推荐if-else

switch (day)
{
    case 0:
        dayName = "sunday";
        break;
    case 1:
        dayName = "Monday";
        break;

    default :
        dayName = "Unknown";
        break;
}

数据用法¶

day = 0
switcher = {
    0: 'Sunday',
    1: 'Monday',
    2: 'Tuesday',
}
# 字典映射
# 没有实现default
day_name = switcher[day]
print(day_name)
day_name_2 = switcher.get(6, 'Unknown')
print(day_name_2)

函数用法¶

# 利用函数

def get_sunday():
    return 'Sunday'
def get_monday():
    return 'Sunday'
def get_tuesday():
    return 'Sunday'
def get_default():
    return 'Unknown'

switcher = {
    0: get_sunday,
    1: get_monday,
    2: get_tuesday,
}
day = 2
day_name = switcher.get(day, get_default)()
